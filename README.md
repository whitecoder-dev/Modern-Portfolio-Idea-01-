# Interactive Rotating Social Hub 🚀 | Portfolio Concept 01

A premium, minimalist, and highly interactive social navigation hub designed for modern portfolios. This project focuses on fluid motion, glassmorphism, and a seamless user experience.

## ✨ Overview
This is a creative approach to personal branding, moving away from static links to a dynamic 360° rotating menu. Built with a focus on "Awwwards-level" aesthetics and smooth transitions.

🔗 **Live Demo:** [View Project](https://whitecoder-dev.github.io/demos/rotate_portfolio/)

## 🚀 Features
- **360° Interactive Menu:** Smooth rotation around a central profile anchor.
- **Dynamic Theming:** Background colors and accents shift based on user interaction.
- **Glassmorphism UI:** Modern, translucent navigation bar and elements.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
- **Minimalist Design:** Clean typography and high-end "Apple-style" spacing.

## 🛠️ Tech Stack
- **HTML5** - Semantic structure.
- **CSS3** - Advanced animations, Flexbox/Grid, and Glassmorphism effects.
- **JavaScript (ES6+)** - Logic for rotation and dynamic interactions.
- **GSAP (Optional/Recommended)** - For ultra-smooth motion curves.
